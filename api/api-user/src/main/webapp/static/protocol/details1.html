精英贷

专门为拥有稳定工作及收入的中高端授薪人群推出的信贷方案。

期         数： 12、18、24、36期

借款额度： 最高20万元

还款方式： 按月等额本息还款

申请标准

年龄在22-59周岁的中国大陆公民；

申请人在现单位连续工作3个月(不含试用期);

所需材料

个人身份证、征信报告、工作证明、收入证明。

汇优贷

期         数： 12、18、24期

借款额度： 最高20万元

还款方式： 按月等额本息还款

申请标准

年龄在22-59周岁的中国大陆公民；

申请人在现单位连续12个月缴纳社保或公积金凭证；

月收入不低于4000元；

所需材料

个人身份证、征信报告、工作证明、收入证明、个人住址证明。

助业贷

专门为经营类人群推出的信贷方案。

期         数： 6、9、12、18、24期

借款额度： 最高20万元

还款方式： 按月等额本息还款

申请标准

年龄在22-59周岁的中国大陆公民；

申请人须是经营主体的法人或持股20%以上股东；

申请人企业经营2年以上；

所需材料

个人身份证、征信报告、工作证明、收入证明、个人住址证明、经营地址证明、经营证明。

医师贷

专门为职业为医生或教师的授薪人群推出的信贷方案。

期         数： 12、18、24、36期

借款额度： 最高20万元

还款方式： 按月等额本息还款

申请标准

年龄在25-55周岁的中国大陆公民；

申请人在现工作单位连续工作满6 个月(含试用期)；
