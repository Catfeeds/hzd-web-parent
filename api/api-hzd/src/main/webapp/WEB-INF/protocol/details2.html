移交类一期

专门为本人名下有车辆的人群(车辆需办理质押)，推出的期限一个月、可展期的抵押贷款方案。

期         数： 1期(1个月)，展期最高5次

借款额度： 不超过车辆评估价值的90%

还款方式： 到期一次性还本付息

申请标准

年龄在18-60周岁的中国籍（不含港澳台）公民，具有完全民事行为能力；

本人名下拥有在当地车管所登记的国产或进口的非营运性质机动车，即借款人须拥有对该车辆的所有权。

所需材料

身份证、机动车登记证、机动车行驶证、本人银行借记卡、车辆钥匙。

汇车贷

专门为本人名下有车辆的人群(车辆需办理抵押并加装GPS)，推出的抵押贷款方案(分长期和短期)。

期         数： 短期方案(3期、4期、5期、6期，1个月为一期)、长期方案(9期、12期、18期、24期，1个月为一期)

借款额度： 不超过车辆评估价值的80%

还款方式： 等额本息

申请标准

年龄在22-59周岁的中国籍（不含港澳台）公民，具有完全民事行为能力；

本人名下拥有在当地车管所登记的国产或进口的非营运性质机动车，即借款人须拥有对该车辆的所有权。

所需材料

短期方案：身份证、机动车登记证、机动车行驶证、车辆钥匙、征信报告、固定地址证明、车险保单、本人银行借记卡、通话详单；

长期方案：身份证、机动车登记证、机动车行驶证、车辆钥匙、本人驾照、征信报告、居住证明、工作证明、车险保单、本人银行借记卡。
