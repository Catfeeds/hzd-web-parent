有所不为 有所不畏——做合规发展的领跑者

2016年10月27日

日前，在连续三年蝉联全美银行市值第一后，2016年9月，富国银行把头牌拱手送给了摩根大通，起因是一张1.85亿美元的天价罚单。据美国消费者金融保护局的指控，富国银行在开展交叉销售业务的过程中，私自为用户开了数百万个虚假账户。这一令人震惊的事实，引发了人们对于富国银行“交叉销售”业务模式的质疑。

富国银行一直贯彻的理念是“以客户为中心，而不是以产品为中心”，显然，账户造假事件已经清晰地展示富国银行违背了初衷。深思，富国银行不惜突破道德底线，急功近利的行为，再清楚不过地说明今天的银行业早已被功利和激进挟持，合规和稳健经营已被抛诸脑后。

所以，在合规文明的建设上，企业必须将合规的理念延伸与穿透组织的每一个层面，在管理者率先垂范的同时，让合规基因渗透到每个员工的服务举止中。

有所不为  脚踏实地智者之选

2016年作为互联网金融规范元年，互联网金融正迎来发展的春天。

国务院日前发布的《互联网金融风险专项整治工作实施方案》与2015年十部委联合发布的《关于促进互联网金融健康发展的指导意见》以及《网络借贷信息中介机构业务活动管理暂行办法》的要求和逻辑都是一致的，旨在为帮助互金领域建立和完善监管，促成扶优限劣的环境，促进行业健康可持续发展。

这对整个行业来说是重大利好，同时也更加坚定了汇中坚持合规发展，实现“普汇大众 财富中华”的信心。

汇中自成立之初，就注重保护广大客户的合法权益。汇中始终把客户的利益放在首位，为客户积极构建诚信、安全、透明的财富增值体系。同时拥有完善的风控体系和行业一流的精英团队，能独立、客观、公正的为客户定制专属资产配置方案，并对客户的财富规划方案执行全程跟踪，及时规避市场风险，最大限度的保障客户的财产安全。

2016年，是网贷行业规范发展的一年，也是汇中的合规年、转型年、精细化管理年。监管政策的落地，规范了行业的发展方向，避开风险，有所不为，脚踏实地才是智者之选，只有坚持职业操守和道德操守、严格的操作规范，才能守住发展的生命线，推动普惠金融事业的持久发展。

有所不畏  合规发展携梦起航

不得不说，国家出台一系列的法规、管理办法、措施，对互联网金融行业的发展来说是一大利好。监管逐渐落地之后，不少专家建言，一要有效鉴别互联网金融公司的属性并预制相应的监管，二要从制度建设层面加强投资者教育，三要从业机构要认清金融本质，加强风控水平。

其实，早在监管意见出台之前，汇中就严苛定位自身平台，走在合规发展之路的前列。在政策落地后，汇中又积极启动执行各项合规举措。从领导高管到基层员工，深刻解读制度精神，积极对接监管部门；加入首都金融服务商会等各类行业组织，寻求进一步规范途径；并制定合规事项行事历，逐一进行对标，同时积极对接第三方机构，寻求全面合规方案；始终加强全员的培训与宣传。突破种种局限，有所不畏，真正将合规精神执行到每一项工作中。

作为全球化综合金融企业，汇中一直致力于打造合规、安全、合法的平台。愿通过自律规范与行业领导力助推行业健康发展。

未来，汇中将实现公司整体业务布局的转型升级，成长为一家以综合金融为主，产融结合的综合金融企业。希望在合法合规的经营下实现稳健的发展，通过结合国际资本开启全球资产配置咨询服务模式，为行业的稳健发展提供更为有利的引导与保障。
