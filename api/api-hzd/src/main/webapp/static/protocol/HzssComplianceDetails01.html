因为合规，所以安全

2016年11月15日

作为新兴金融模式，互联网金融在发展初期处于摸索阶段，没有现成路子可走。初期建立的一些平台难免呈现野蛮生长的态势，暴露出来一些风险因素。事实上，随着监管措施的落地，互联网金融更加安全，前景向好。据最新数据，P2P网贷行业历史累计成交量已突破3万亿，投资者近500万人！

对于真正有志于创新的互联网金融平台来说，因为有监管，所以更安全。汇中响应国家发展普惠金融的动议，积极探索互联网金融创新模式，是真正意义上的互联网金融创新平台。早在2013年，汇中就严格自律，坚持创新与安全并重，在行业早期的乱局中，如一股清流，不忘初心，坚持创新与安全并重。汇中认为“创新是一切发展不竭的动力和源泉。但对于金融投资领域来说，更需要注意伴随着创新变革而来的也可能是难以预料的各类风险”。

在风险控制的技术创新层面，汇中积极探索，构建了基于大数据技术的一套全方位风控系统。在风险控制的模式创新层面，汇中自发展之初就严格自身平台的定位，甚至在监管政策出台之前汇中的一些做法即与监管政策相符合。这充分说明，汇中一直以来都走在行业风险控制和合规发展的前沿，顺应了互联网金融的趋势。

中国人民银行行长周小川在华盛顿参加国际货币基金组织时强调，中国正致力于“推动互联网金融平台建设，以降低中国金融风险”。合规化运营是推动互联网金融平台建设的重要目标。规范是安全的保障，规范是为了更好的发展。汇中深刻认识到，各项监管细则的出台，明确了互金行业各项业务合法与非法、合规与违规的边界，为真正的互联网金融创新平台的发展指出明路，对合法合规的行为给予保护和支持，是帮助互金平台进行风险控制的具体指导。所以当各项合规标准和监管政策出台之后，汇中积极启动执行各项合规举措，从领导高管到基层员工，深刻解读制度精神，更加明确了平台的信息中介性质，同时积极对接第三方机构，完成资金的银行存管，逐一进行对标，全面落实各项合规标准。这些是汇中能够不断创新，安全可持续发展的保障。

据《每日经济新闻》分析,随着各项监管政策逐渐落地，行业发展愈加规范，网贷信息中介平台处于优胜劣汰、寡头浮现的状况，并且表现出集中度进一步提高的现象，同时，弱小平台的良性退出，成为行业趋势。风险控制做的好不好，够不够安全将是决定互联网金融平台会不会在这轮洗牌中被淘汰的关键因素。汇中人在坚持安全运营的同时，始终不忘行业本质，秉承普惠金融的初心，不断地尝试技术创新、服务创新，尽最大努力使传统金融没有覆盖的人群能够享受到金融服务。我们相信，坚持合规发展的汇中必将在新一轮的行业洗牌中脱颖而出。
