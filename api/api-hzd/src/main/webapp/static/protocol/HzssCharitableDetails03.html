【汇中正能量】西安营业部李彤彤等伙伴夏日展业挽救弃婴，见证温暖行动

2016年6月30日

世上只有爸妈好，有妈的孩子像个宝，离开妈妈的怀抱，幸福哪里找。这样耳熟能祥的歌曲，我们是否深有体会呢？

试想一下，如果当我们身亲经历的时候，该是那么伤心和无助。

2016年6月27日，便有一位美女体会到了这般滋味。这是发生在西安营业部的李彤彤和她的小伙伴们在晚上展业的路上，途中意外发现一个弃婴。因始发地就在医院附近，当即求助医院并进行报警，民警到场之后李彤彤和小伙伴们前前后后一直协助照顾好几个小时，最终民警联系了接收医院，才放心的离开。

这样的正能量值得我们去学习，为他们的精神我们感到喝彩，身为汇中人我们感到骄傲。

大家都知道《等着我》是中央电视台综合频道大型公益寻人栏目，明星，志愿者等等都在帮助全国人民踏寻寻亲之路。宝宝是妈妈十月怀胎生下的天使，宝宝是爸妈的小棉袄，今天我们遇见了，就应该伸出援助之手，但是我们希望宝可以健康成长，如果宝宝的爸妈看见了，请速来寻找自己的宝贝，他们希望父母的陪伴。同时，我们也希望这个天使可以健康的成长。
