企业简介

致力为小微企业、工薪阶层提供推荐借款信息咨询服务，进行合理全面的财务规划，量身设计最优的借款解决方案，为他们的发展成长提供助力。

向客户提供的融资方案有:
工薪信用类(精英贷、汇优贷、汇薪贷、汇薪汇楼贷、医师贷、公积金贷)；
经营信用类(助业贷、助业汇楼贷)；
抵押类(汇车贷-GPS、汇车贷-移交)。

已在全国设立7个大区，70多家分支机构，服务覆盖24个省，66个大中型城市，拥有近三千名借款信息咨询专业团队。
