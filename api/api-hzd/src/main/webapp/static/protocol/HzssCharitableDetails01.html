予人玫瑰 手留余香——贺“汇·爱”暖冬捐助活动圆满收官

2016年10月24日

“汇·爱”暖冬捐助青海玉树阿永寺日玛寄宿小学活动完美收官啦！！

活动自9月26日发起以来，得到了社会各界爱心人士及汇中家人的大力支持，历时28天累计募集爱心善款107861元，远超定向捐助阿永寺日玛寄宿小学的十万元爱心捐款。

聚沙成塔，集腋成裘，滴滴爱心之水，终于汇聚成了温暖的江流。感谢所有人的善举，让阿永寺日玛寄宿小学的孩子们的冬天不再寒冷。

本次活动作为“汇·爱”系列社会责任活动之一，秉承奉献爱心、传播正能量的理念，旨在为青海玉树阿永寺日玛寄宿小学的孩子们送温暖、送关爱，为社会的健康美好发展奉献一份力量。

募捐活动一经开启，汇中所有的小伙伴便积极行动起来。汇中财富积极动员大家献爱心，汽车金融事业部的同事连续两次组织募捐，公司其他各职能端也都积极行动。

其中，汇中盛世的伙伴们在了解阿永寺日玛小学的情况后，不仅在汇中盛世管理中心自发组织捐款，还发表倡议书号召全国各大区的伙伴开展“汇·爱”暖冬捐助活动，在汇中总裁助理马海涛领导下，在各营销区域领导的大力支持下，短短几日，汇中盛世全体伙伴共为青海玉树阿永寺日玛寄宿小学募捐爱心善款 76484元。

汇中全体伙伴们表示，在此次捐助活动中，我们都拥有一个共同的愿望，希望孩子们能拥有一个温暖的冬天，希望我们的爱心能够一直传递下去，让爱的火焰永不熄灭！

孩子是祖国的未来，一个温暖舒适的学习环境是汇中送给阿永寺日玛寄宿小学孩子们最好的礼物。 一直以来，汇中都秉持着“普汇大众 财富中华”的使命，始终以关爱帮助社会各界人士为己任，积极投身公益事业，并启动“汇·爱”系列社会责任活动，旨在奉献爱心、传播正能量，让更多的人在“汇·爱”系列活动中受益 。

为阿永寺日玛寄宿小学的孩子们筹款修葺校舍，正符合汇中一直以来的价值观，也是一家企业应当承担的社会责任。希望我们的绵薄之力，能帮助孩子们成长得更好一些，将来为社会创造更多的幸福！也希望通过企业的力量让我们的社会变得更加温暖与美好。
