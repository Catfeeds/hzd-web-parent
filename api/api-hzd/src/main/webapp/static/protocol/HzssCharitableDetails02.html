点点滴滴热血浓 人道博爱处处情——汇中伙伴无偿献血，用爱心为生命加油

2016年10月24日

汇中盛世重庆万州一部客户经理蔡成彦的父亲因胃溃疡复查血常规，发现血小板总数仅为4(10³/L)，而正常值范围在125—130之间，医院诊断为原发性急性血小板减少症，病危。可偏偏就在这紧急时刻，万州血站O型血小板告急。无奈之下，他在朋友圈发布了一条求助消息。

看到他的消息之后，同事们纷纷将此消息进行了转发，营业部经理张文辉先生第一时间在群内倡导大家一起助力为生命加油。同为O型血的何淼、王毅、易绍雄三人与蔡成彦一起去到万州血站进行了无偿献血。捐一份热血，献一片爱心，帮助同事渡过难关，帮助其父亲战胜病魔。

同事们献血后都放弃休息，坚持上班，这种无私奉献、助人为乐的精神让人肃然起敬。正如歌中所唱：“只要人人都献出一点爱，世界将变成美好的人间。”通过这样的行动，将爱心传播出去，为生命加油助力。

赠人玫瑰，手留余香。通过他们的爱心行动，给蔡成彦父亲带去生的希望，带去爱的阳光！

正能量满满的汇中，团结有爱的汇中家人，鼓舞着每一个人用心工作热爱生活，踏实前进，共享生命的感动！

同时，汇中勇担社会责任，积极投身公益事业，将爱心慈善事业放在越来越重要的位置。并通过开展“汇·爱”系列公益活动，累计帮助近万人，用实际行动弘扬慈善精神，践行了普惠金融机构为构建和谐社会贡献力量的承诺。
